## API created platform : http://localhost:3000/convert?from=USD&to=INR&amount=10

## Currency Converter API (Backend)

    * A simple backend application built using Node.js, Express, and Axios that converts currency values in real-time using an external exchange rate API.

    * The API accepts query parameters such as from, to, and amount, fetches the latest exchange rates using an API key, and returns the converted currency value along with the exchange rate.

    * It demonstrates real-time API integration, query handling, error handling, and JSON response formatting.

## Endpoint
   >GET /convert
## Example:
   >http://localhost:3000/convert?from=USD&to=INR&amount=10

## Features
   >Real-time currency conversion 💱
   >External API integration using API key 🔑
   >Query parameter handling
   >Error handling for invalid inputs
   >Clean JSON response

## Tech Stack
   >Node.js
   >Express.js
   >Axios
